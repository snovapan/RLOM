# 儒象岛®

## 简介

**儒象岛®** 逆向物流信息管理系统，是您便捷、高效管理退货逆向物流的好助手。

**儒象商城** 作为儒象岛®的一部分，展示了退货商品再次销售的流程，帮助企业将退货重新转化为利润。


## 项目目录

这个目录包含了儒象商城的后台系统，使用Ruby on Rails作为框架。
